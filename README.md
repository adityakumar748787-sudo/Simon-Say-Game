# 🎮 Simon Says Game

## 📌 Overview

The *Simon Says Game* is a classic memory-based game where players must repeat a sequence of colors or sounds in the correct order. With each round, the sequence becomes longer, challenging the player's memory and concentration.

This project is a web-based implementation of the Simon Says game built using modern frontend technologies.

---

## 🚀 Features

* 🎯 Interactive gameplay with increasing difficulty
* 🎨 Color-based visual cues
* 🔊 Sound effects for each button
* 📈 Score tracking system
* 🔁 Game restart functionality
* 💡 Responsive design for different screen sizes

---

## 🛠️ Tech Stack

* **HTML5** – Structure of the game
* **CSS3** – Styling and animations
* **JavaScript (Vanilla JS)** – Game logic and interactivity

---

## 🎮 How to Play

1. Press any key or click the start button to begin the game.
2. The game will highlight a color in a sequence.
3. Repeat the sequence by clicking the buttons in the same order.
4. Each correct round adds a new step to the sequence.
5. If you make a mistake, the game ends.
6. Try again and beat your high score!
